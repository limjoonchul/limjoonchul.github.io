문제

상속을 사용한 고객관리 연습문제

  • 고객의 등급에 따라 할인율, 적립금을 다르게 적용한다.

  • Customer 클래스를 만든다.

  • 자식클래스인 VIPCustomer, GoldCustomer클래스를 만든다.

  • Customer클래스의 상속을 받는 VIPCustomer클래스에는 기능을 추가한다.

    • 제품 구매시 10% 할인

    • 보넛 포인트 5% 적립

    • 담당 상담원 배정

문제 구현

  • Customer 클래스

  • VIP Customer 클래스

  • Test 클래스

  • Overriding Test 클래스
