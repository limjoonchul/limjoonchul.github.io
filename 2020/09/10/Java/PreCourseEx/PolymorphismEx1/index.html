PreCourse-PolyhorphismEx

문제

배열을 활용하여 상속과 다형성 구현하기

  • 고객의 등급에 따라 할인율, 적립금을 다르게 적용한다.

  • Customer 클래스를 만든다.

  • Customer 클래스의 상속을 받는 VIPCustomer클래스에는 기능을 추가한다.

    • 제품 구매시 10% 할인

    • 보넛 포인트 5% 적립

    • 담당 상담원 배정

  • Customer 클래스의 상속을 받는 GoldCustomer클래스를 만든다.

  • VIP고객과 일반고객의 중간 등급의 고객

    • 제품을 구매시 10%할인을 해주는 것은 같다.
    • 보너스 포인트는 2% 적립해준다.
    • 담당 상담원은 없다.

문제 구현

  • Customer 클래스

  • VIPCustomer 클래스

  • GoldCustomer 클래스

  • Test 클래스
