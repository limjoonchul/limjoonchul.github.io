JavaBasic_InputOutput

자료의 입출력(Inputs and Outputs)

출력 메소드

  • 포맷 문자열을 이용한 문자열/기본형 출력
  • 객체를 직접 출력 -> toString() 재정의를 통해 어떻게 출력할 것 인지를 구현할 수 있다.

printStream의 메소드

  • print():

    • public void print(Object obj)
    • obj를 스트림으로 출력한다.

  • println():

    • public void println(Object x)
    • x를 스트림으로 출력하고 줄바꿈한다.

  • printf() :

    • public PrintStream printf(String format, Object … args)

    • Format 맞추어 args를 출력한다.

    • 정수 포맷팅

    • 실수 포맷팅

      입력 메소드

  • Scanner 클래스를 이용하여 입력 받을 수 있다.

  • System.in은 키보드 입력을 받는 것이다 .PC에서 사용하기 때문에 표준입력을 이것을 사용하는 것이다.

  • Scanner 클래스의 주요 메소드

    • next(): 공백을 기준으로 한 단어씩 입력 받는다.
    • nextLine(): 한 줄 전체를 입력 받는다. (enter로 구별)
    • nextInt(): (공백을 기준으로) int 값을 입력 받는다.
    • nextDouble : double 값을 입력 받는다.
    • close() : 입력 스트림을 종료한다.
      • close() 를 사용하지 않으면 메모리 낭비가 될 수 있다,
        gc가 제대로 동작이 되지 않았을 때 사용을 신경써 주는게 좋다.
    • 값을 여러개 입력받을 때, nextInt()는 공백을 기준으로 값을 입력받기 때문에, 만약 10이라는 값을 입력하고 enter를 누르면
      10이라는 값은 출력이 되는데, enter도 하나의 값이여서 값이 남아있게 된다. 그래서 sc.nextLine()으로 enter 값을 입력받게 해게 해서
      없애준 다음에 다음 값들을 변수에 담아 주도록 해야 한다.
